
1. Setup your Termux using below cmds. Skip setup if you had already did it before.

2. **Clone the Repository:**
   ```
   git clone https://github.com/mouricedevs/telegram-bot.git
   ```
   ```
   cd telegram-bot
   ```
   
3. **Install Dependencies**:
   ```
    npm install 
   ```

4. **Running the Bot**
   ```
    node gifted.js 
   ```
